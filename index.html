OBS PRO FÚTBOL

La configuración es de un OBS, diseñado para la retransmisión de partidos.

Estructura de Escenas

La organización de escenas es clave para un manejo eficiente durante la retransmisión:

  • [S] PROD:Es la escena principal que se envía a la salida de programa (la que ve el espectador).
  • [S] CÁMARAS:Contiene todas las escenas de cámaras individuales anidadas. El orden se gestiona con las teclas F1-F12 para cambiar la cámara activa.
  • [S] SONIDOS:Agrupa todas las fuentes de audio (micrófonos de comentaristas, sonido ambiente, etc.).
  • [C] Cámara 01, 02...:Cada cámara (c1, c2, etc.) tiene su propia escena individual para facilitar su manejo, grabación aislada y monitorización.

Atajos de Teclado

  • Mostrar MarcadorCtrl + M
  • Ocultar MarcadorCtrl + N
  • Marcador GrandeCtrl + B
  • Mostrar Cámaras (1-12)F1 - F12
  • Grabar Cámaras IzquierdaCtrl + I
  • Grabar Cámaras DerechaCtrl + O
  • Guardar Repetición (8s)Ctrl + G

Monitorización

Para monitorear todas las cámaras simultáneamente, se utiliza la herramienta de OBS:

Herramientas > durchblick

Repeticiones

La grabación de repeticiones consume muchos recursos. Con 12 cámaras, se recomienda grabar solo 6 para mantener un uso de GPU estable (probado con RTX 3070 Ti).

Overlay

La configuración del marcador y plantillas se encuentra en:

app.overlay.uno

Conexión de Cámaras (SRT)

OBS (Receptor)

En la fuente multimedia de OBS, usa esta dirección. Cambia <ID> por el de la cámara.

srt://0.0.0.0:22222?mode=listener&streamid=<ID>

Dispositivo Emisor (Móvil, Veo, etc.)

  1. En la app de streaming, crea una nueva conexión.
  2. Configure la URL y Puerto:
    • Red Local (WiFi): 192.168.1.100:22222
    • Red Externa (Internet): tu.dominio.externo:22223
  3. Modo SRT: Caller
  4. Latencia: 1000 (1 seg).
  5. Stream ID: <ID> que corresponda.
